## Overview

Hybrid cloud network architecture — on-premise to AWS, Azure, GCP with Aviatrix unified fabric. This project is part of the MAIA — Multi-Cloud AI Architect portfolio, providing reference architectures and reusable building blocks for cloud architects working across AWS, Azure, and GCP.
